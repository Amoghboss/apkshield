## 🔧 Technical Architecture

### Backend (Python)
```
FastAPI + Androguard + WebSockets
├── Permission risk database (35+ patterns)
├── Fraud detection algorithms (8 patterns)
├── Certificate validation
├── Component analysis
└── Real-time progress streaming
```

### Frontend (React)
```
React 19 + TypeScript + Tailwind CSS 4
├── APKProtectionTool (main scanner)
├── Auth component (login/signup)
├── WebSocket client
├── Share functionality
└── Responsive design
```

### Data Flow
```
User Upload → FastAPI → Androguard → Risk Analysis → WebSocket → React UI
```
