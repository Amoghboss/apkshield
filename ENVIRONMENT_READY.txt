🎯 WHAT YOU NEED TO DO (2 MINUTES):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open: frontend/.env.local

2. Replace these lines:
   VITE_SUPABASE_URL=YOUR_SUPABASE_PROJECT_URL_HERE
   VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY_HERE

3. With your actual Supabase credentials:
   VITE_SUPABASE_URL=https://xxxxxxxxxxxxx.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.your_key_here

4. Save the file

5. Restart frontend server:
   cd frontend
   npm run dev

📍 WHERE TO FIND YOUR CREDENTIALS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://supabase.com/dashboard
2. Select your project
3. Click: Settings → API
4. Copy:
   • Project URL → Use as VITE_SUPABASE_URL
   • anon public key → Use as VITE_SUPABASE_ANON_KEY

🔐 OPTIONAL: ENABLE AUTHENTICATION PROVIDERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

In Supabase Dashboard:

1. Go to: Authentication → Providers
2. Enable: Email (for email/password sign-in)
3. Enable: Google (for Google OAuth - requires Google Cloud setup)
4. Click: Save

5. Go to: Authentication → URL Configuration
6. Add redirect URLs:
   • http://localhost:3000
   • http://localhost:3001
   • http://localhost:3002
   • http://localhost:3003
7. Click: Save
